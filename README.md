# Spatial-Network-and-Multi-Objective-Optimization-for-Mall-Store-Layout-Planning
本项目以广州太古汇多层购物中心为实例，搭建融合空间网络建模、智能体客流仿真与多目标进化优化的商场店铺布局规划模型。项目先将商场楼层通道、扶梯、铺位构建拓扑空间网络，采用分时泊松客流结合改进 Huff 吸引力模型，区分多类消费人群完成全天行走、消费行为 ABM 仿真，量化各铺位客流潜力与基础收益。以商场总营业额为单目标，使用模拟退火算法求解最优铺位分配方案，相较原始布局营收提升 47.66%。进一步综合商场收益、商户公平性、顾客步行体验、业态均衡四大冲突目标，基于 NSGA-II 算法生成帕累托最优解集，筛选多方均衡的折中布局方案。多组参数灵敏度实验验证模型稳定可靠，可为商业地产商场招商、业态空间调整提供量化、可落地的规划决策工具。
