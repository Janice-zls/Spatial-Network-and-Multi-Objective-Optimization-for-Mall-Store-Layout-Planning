# Spatial-Network-and-Multi-Objective-Optimization-for-Mall-Store-Layout-Planning
本项目以广州太古汇多层购物中心为实例，搭建融合空间网络建模、智能体客流仿真与多目标进化优化的商场店铺布局规划模型。项目先将商场楼层通道、扶梯、铺位构建拓扑空间网络，采用分时泊松客流结合改进 Huff 吸引力模型，区分多类消费人群完成全天行走、消费行为 ABM 仿真，量化各铺位客流潜力与基础收益。以商场总营业额为单目标，使用模拟退火算法求解最优铺位分配方案，相较原始布局营收提升 47.66%。进一步综合商场收益、商户公平性、顾客步行体验、业态均衡四大冲突目标，基于 NSGA-II 算法生成帕累托最优解集，筛选多方均衡的折中布局方案。多组参数灵敏度实验验证模型稳定可靠，可为商业地产商场招商、业态空间调整提供量化、可落地的规划决策工具。
## 一、四大核心研究模块
### 模块1：商场空间网络构建与顾客ABM仿真
1. 将商场楼层、通道、扶梯、商铺抽象为无向空间拓扑网络图，利用NetworkX完成节点、通行边建模；
2. 采用分时泊松分布模拟早中晚分时段客流，划分购物、餐饮、家庭、休闲四类顾客智能体；
3. 融合BFS最短路径算法与改进Huff吸引力模型，模拟顾客选店、移动、消费全过程；
4. 输出各铺位分时段客流热力、单日模拟营业额、区域拥堵程度等基础评价指标。

### 模块2：基于模拟退火的单目标营收最大化优化
1. 建立商铺-铺位匹配收益矩阵，加入楼层适配、面积、业态互补约束惩罚项；
2. 针对大规模铺位组合NP难问题，采用多起点模拟退火+局部邻域搜索加速收敛；
3. 在不改动电梯、卫生间、出入口等公共设施前提下，求解全局营收最优布局；
4. 对比原始商场布局，优化方案总营业额提升47.66%。

### 模块3：基于NSGA-II的四目标多目标布局优化
1. 确立四大优化目标：商场整体营业收入、商户平均利润、商户收益基尼系数（公平性）、顾客空间出行满意度；
2. 通过非支配排序、拥挤距离筛选生成帕累托最优解集，不存在单一布局同时满足全部目标最大化；
3. 采用熵权法对多目标指标加权打分，筛选兼顾多方利益的折中推荐布局方案。

### 模块4：模型参数灵敏度鲁棒性检验
1. 扰动顾客到达率、消费转化率、店铺吸引力衰减系数、多目标权重四类核心参数；
2. 对比不同参数场景下优化布局的收益、公平、体验指标波动幅度；
3. 验证模型在客流规模、消费偏好发生变动时，优化结论仍具备稳定优势。

## 二、技术栈与核心算法
### 开发环境
Python 3.9+ | Pandas | NumPy | Matplotlib | NetworkX | Scipy
### 核心算法清单
1. 空间拓扑网络建模、BFS最短路径路径规划
2. ABM多类型智能体仿真、分时泊松客流生成、改进Huff选址吸引力模型
3. 多起点模拟退火启发式组合优化
4. NSGA-II非支配排序遗传多目标优化、帕累托前沿解集筛选
5. 基尼系数公平性度量、熵权法多指标综合评价

## 四、项目运行指南
1. 安装依赖库
```bash
pip install pandas numpy matplotlib networkx scipy
```
2. 数据准备：楼层拓扑、商铺业态基础数据内置在code脚本中，可按需替换其他商场数据集；
3. 执行顺序（存在数据依赖，不可颠倒）：
   ① 运行01生成客流仿真基础数据 → ② 运行02得到营收最优单目标布局 → ③ 运行03生成多目标最优解集 → ④ 运行04完成模型稳定性测试；
4. 所有可视化图表、数值评价报表自动输出至output文件夹。

## 五、核心实验成果
1. ABM仿真：完整模拟单日一万四千余名顾客移动消费，精准还原餐饮午晚高峰客流分布特征；
2. 单目标优化：原始布局日营业额748.16万元，优化后可达1104.72万元，增幅47.66%；
3. 多目标折中方案：兼顾高营收、低商户贫富差距、低顾客步行损耗，综合均衡性最优；
4. 灵敏度测试：客流、消费参数大幅波动下，优化布局收益提升优势稳定，模型鲁棒性良好。

## 六、模型优势与局限
### 创新优势
1. 真实还原多层商场空间通行约束，拓扑网络贴合实体建筑结构；
2. 多人群分时智能体仿真，客流模拟更贴近线下真实消费行为；
3. 启发式算法高效求解大规模铺位组合优化，规避暴力枚举；
4. 多目标均衡方案同时兼顾商场运营方、入驻商户、到店顾客三方利益，落地实用性强。
### 现存局限
1. 缺乏商场真实POS消费、租金签约原始数据，部分行为参数采用行业经验值；
2. 未考虑节假日促销、多人结伴购物、临时拥堵等复杂动态场景；
### 后续拓展方向
接入线下客流摄像头真实数据校准仿真参数；新增节假日临时摊位、动态招商周期优化模块。
